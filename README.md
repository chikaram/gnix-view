## How to use

You need:

  - ZendFramework (a newer one)
  - PHP >= 5.2

    $view = new Gnix_View_AutoEscaper();
    Zend_Controller_Action_HelperBroker::getStaticHelper('viewRenderer')->setView($view);

That's all!
